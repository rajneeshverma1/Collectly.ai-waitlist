# Collectly.ai Waitlist

A high-conversion landing page for Collectly.ai, built with Golang and modern Vanilla CSS.

## Features
- **Modern Tech Aesthetic**: Sleek grid backgrounds and glassmorphism.
- **Interactive CTAs**: Supports `Cmd+S` and `Cmd+D` shortcuts.
- **Responsive**: Fully optimized for mobile and desktop.
- **AI-Driven Preview**: High-quality product mockup generated via AI.

## Getting Started
### Prerequisites
- Go 1.18+

### Running the Project
```bash
make run
```
The application will be available at `http://localhost:8080`.

### Building for Production
```bash
make build
```
